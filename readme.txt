BP xProfile Username Field adds a username drop down selector to Extended Profiles in BuddyPress.

One you have installed and activated this plugin you will see a Username Selector option as a profile field.

Note the field value will store the login name of the selected user.

== Description ==

The BP xProfile Username Field plugin adds a username drop down selector to Extended Profiles in BuddyPress.


== Installation ==

1. Extract the plugin archive
1. Upload plugin files to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
